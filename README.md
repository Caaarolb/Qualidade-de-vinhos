# 🍷 Wine Quality Analysis

This project explores the **Wine Quality Dataset** using Python for data cleaning, exploratory data analysis (EDA), statistical analysis, and data visualization. The objective is to identify the factors that influence wine quality and extract insights from physicochemical characteristics.

## 📌 Project Overview

Wine quality is influenced by several chemical properties. In this project, I analyzed a real-world dataset to understand the relationships between these variables and the quality score assigned to each wine.

The analysis includes:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical summaries
- Correlation analysis
- Data visualization
- Insights and conclusions

## 📊 Dataset

The dataset contains physicochemical measurements of wines, including:

- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol
- Quality (Target Variable)

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 📈 Analysis Performed

- Data inspection
- Missing value verification
- Descriptive statistics
- Correlation matrix
- Distribution analysis
- Feature relationship analysis
- Data visualization with charts
- Identification of variables related to wine quality

## 🔍 Key Insights

- Alcohol content shows a positive relationship with wine quality.
- Volatile acidity tends to negatively impact quality.
- Most wines are classified as medium quality.
- Several physicochemical variables exhibit moderate correlations with the target variable.

# 🍷 Análise da Qualidade de Vinhos

Este projeto tem como objetivo analisar um conjunto de dados sobre a qualidade de vinhos utilizando **Python** para limpeza de dados, análise exploratória (EDA), estatística descritiva e visualização de dados. O foco é identificar quais características físico-químicas estão mais relacionadas com a qualidade final dos vinhos.

---

## 📌 Objetivo

Realizar uma análise exploratória dos dados para compreender o comportamento das variáveis, identificar padrões, correlações e gerar insights que auxiliem na interpretação dos fatores que influenciam a qualidade dos vinhos.

---

## 📊 Base de Dados

O conjunto de dados contém informações sobre diversas propriedades físico-químicas dos vinhos, como:

- Acidez fixa
- Acidez volátil
- Ácido cítrico
- Açúcar residual
- Cloretos
- Dióxido de enxofre livre
- Dióxido de enxofre total
- Densidade
- pH
- Sulfatos
- Teor alcoólico
- Qualidade (variável alvo)

---

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📈 Etapas da Análise

- Importação da base de dados
- Limpeza e preparação dos dados
- Análise exploratória (EDA)
- Estatísticas descritivas
- Análise de correlação entre variáveis
- Criação de gráficos para visualização dos dados
- Interpretação dos resultados e geração de insights

---

## 🔍 Principais Insights

- O teor alcoólico apresenta relação positiva com a qualidade dos vinhos.
- A acidez volátil tende a impactar negativamente a qualidade.
- A maior parte dos vinhos possui classificação intermediária.
- Algumas variáveis físico-químicas apresentam correlação moderada com a variável de qualidade.
